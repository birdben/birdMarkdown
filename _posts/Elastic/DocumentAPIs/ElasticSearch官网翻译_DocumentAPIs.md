## Document APIs

本部分首先介绍Elasticsearch数据复制模型，然后详细描述以下CRUD API：

### Single document APIs（单文档API）

- Index API
- Get API
- Delete API
- Update API

### Multi-document APIs（多文档API）

- Multi Get API
- Bulk API
- Delete By Query API
- Update By Query API
- Reindex API

###### 注意

```
所有的单文档的CRUD API， index参数只能接受单个索引名称或指向单个索引的alias。
```
